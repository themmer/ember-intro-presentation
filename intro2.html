Actions Up Example

Receiver => Controller or Route

        actions: {
          dataChangedAction(selectedData) {
            // Note selectedData was passed from the template for context
            // Do something awesome
            // change route
            // update data
            // swap components by changing conditions
            // etc.
          }
        }

Local Action

  • This will bubble up
  • Order: Controller, Route Hierarchy, then Application Route
  • An error will be produced if no receiver
  • Simple unless you have to go up multiple levels

Closure/Callback Action

  • Passed from controller to template/component
  • {{cover-flow dataList=model selectedData=selectedData dataChangedAction=(action "dataChangedAction")}}
  • Inside coverflow component
  • this.get('dataChangedAction')(selectedData);
